Project Information

  • Category: C# .NET Framework
  • Project Date: 05 November, 2023
  • Project URL: BookShelf BootCamp

An interactive C# application developed for a local library to revolutionize learning of the Dewey Decimal Classification system.

  • Language: C#
  • Environment: Visual Studio
  • Data Structures: Lists, dictionaries, and trees for data management.

  • Transition traditional learning into a digital, interactive experience.
  • Enable users and novice librarians to efficiently identify book categories and specific topic call numbers.
  • Facilitate practical training in shelf organization and management.

  • Interactive Modules: Engaging user experience simplifying the Dewey Decimal System.
  • Search Functionality: Quick topic call number retrieval.
  • Virtual Shelf Organization: Hands-on practice in arranging books.

This software was highly effective in improving learning processes within the library. It was well-received for its contribution to enhancing user engagement and understanding of the Dewey Decimal System.
