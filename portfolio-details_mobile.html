Feather Find

  1. Home
  2. Feather Find Android Application

Project Information

  • Category: Android, Mobile, Developed in Kotlin
  • Project Date: 02 November, 2023
  • Project URL: Feather Find

An intuitive Android Application incorporating multiple API's that offer users the best features in the market. This includes a comprehensive explore section that displays nearby bird species and the sighting frequency throughout the year. This is accompanied by a well designed map showing birding hot spots and more.

  • Language: Kotlin
  • Environment: Android Studio
  • Data Structures: Lists, JSON (Multiple API Integration), MVVM Architecture, Firebase

  • Using E-Bird's API to gather birding hot spot locations.
  • Using Google Maps API to create the map services as well as getting the directions to locations.
  • Allow users to add bird sightings to the map and change between the Imperial and Metric system.

  • Birding Hot spots: Displays hot spot markers on the map allowing users to add a sighting or get directions from their current location.
  • Histogram Bar Chart: Visual bar chart showing sighting frequency of specific species within 50km of the users location.
  • Settings: Users can seamlessly change between the Imperial and Metric system and set a maximum distance to travel.

The application was a success, exceeding the original requirements and delivering a publish ready mobile application. This was a team assignment which consisted of three third year students, in which we received a final mark of 91%.
